核心语法

意大利语名词分单数 singolare和复数 plurale，变复数看词尾。
一本书叫 libro，两本书叫 libri——词尾变了！

单数词尾复数词尾例子
-o-ilibro → libri 书
-a-ecasa → case 房子
-e-icane → cani 狗·阳 / chiave → chiavi 钥匙·阴
🧠

-o变-i，-a变-e，-e全变-i——记住这三条，名词变数不发愁。

📋

词尾规律一览

-o → -i
阳性复数 ✅
libro→libri 书ragazzo→ragazzi 男孩gatto→gatti 猫tavolo→tavoli 桌子vino→vini 葡萄酒
-a → -e
阴性复数 ✅
casa→case 房子ragazza→ragazze 女孩pizza→pizze 披萨macchina→macchine 车scuola→scuole 学校
-e → -i
阴阳都可能 ✅
fiore→fiori 花·阳chiave→chiavi 钥匙·阴cane→cani 狗·阳madre→madri 母亲·阴notte→notti 夜晚·阴dente→denti 牙齿·阳
-ca/-ga → -che/-ghe
阴性加h保发音 ✅
amica→amiche 女友banca→banche 银行collega→colleghe 女同事spiaggia→spiagge 海滩
-co/-go 特殊
看重音位置 🤔
parco→parchi 公园albergo→alberghi 旅馆medico→medici 医生amico→amici 朋友⚠️

那些容易搞混的

不变形词（单复同形）
最常用不规则复数
ifilm电影 ← 不变
ibar酒吧 ← 不变
lecittà城市 ← -à不变
icaffè咖啡 ← 重音不变
icomputer电脑 ← 外来词不变
leuniversità大学 ← -tà不变
gliuomini男人 ← uomo不规则
lemani手 ← mano→mani
leuova鸡蛋 ← uovo阳→阴！
lebraccia手臂 ← -ccio→-ccia
ledita手指 ← -to→-ta
lelabbra嘴唇 ← -bro→-bra
leginocchia膝盖 ← -cchio→-cchia
imila千 ← mille→mila
💡 不变形词单复同形，靠冠词区分单复数。uovo→uova 最特殊：单数阳性，复数变阴性！
