核心语法

形容词必须与所修饰的名词保持性数一致。
意大利语形容词分两大类： 四尾形容词（-o/-a/-i/-e）和 两尾形容词（-e/-i）。
后者无论阴阳性，词尾都一样。

类型单数复数示例
阳性四尾 -o bravo -i bravi ragazzo bravo → ragazzi bravi
阴性四尾 -a brava -e brave ragazza brava → ragazze brave
两尾单数 -e grande -i grandi ragazzo/ragazza grande → grandi
🧠

形容词要变样，跟着名词走：
阳性单数 -o，阴性就换 -a，
复数阳 -i 阴 -e，记牢不出错！
两尾 -e 最简单，变 -i 就是复数。

📋

四尾形容词 -o → -a → -i → -e

bravo
阳性单数 -o
ragazzo bravo
brava
阴性单数 -a
ragazza brava
bravi
阳性复数 -i
ragazzi bravi
brave
阴性复数 -e
ragazze brave
italiano
常用四尾词
libro italiano pizza italiana libri italiani pizze italiane
alto
常见搭配
uomo alto donna alta uomini alti donne alte
📋

两尾形容词 -e → -i，不分阴阳

grande
单数 -e（阳/阴通用）
grande (大) grandi (大·复)
intelligente
词尾不变，只变数
ragazzo intelligente ragazza intelligente ragazzi intelligenti ragazze intelligenti
importante
两尾词示例
esame importante questione importante importante importanti
felice / gentile
常见两尾形容词
felice 快乐 felici gentile 礼貌 gentili

特殊形容词

bello
bello 截尾规则
bel ragazzo ← bello截为bel（辅音前） bello studente ← 不截（s+辅音前） bell'amico ← bell'（元音前） bella donna ← 阴性正常 bei libri ← bei（阳性复·辅音） begli amici ← begli（阳性复·元音） belle donne ← 阴性复数正常
buono
buono 截尾
buon pane ← buon（辅音前） buon vino ← buon（辅音前） buon amico ← buon（元音前也buon！） buona cena ← 阴性正常不变
grande
grande 截尾
gran casa ← gran（辅音前） grand'amico ← grand'（元音前） un grand'uomo ← 伟人
不变色
永远不变！
maglia blu 蓝色上衣 scarpe rosa 粉色鞋子 fiore viola 紫色花

是的，他们非常有趣又聪明
🔑

形容词一定要和名词性数一致！
四尾变四形，两尾只变数，不变色永远不动。

✅ ragazzo bravo ❌ ragazzo brava
✅ donna alta ❌ donna alto
✅ ragazzi intelligenti ❌ ragazzi intelligente
